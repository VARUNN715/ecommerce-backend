# Coding Standards

## Package Naming

- Lowercase
- Singular names

Example

product
category
order

## Class Naming

ProductController
ProductService
ProductRepository

## DTO Naming

CreateProductRequest
UpdateProductRequest
ProductResponse

## Entity Naming

Product
Category
Order

## Method Naming

createProduct()
updateProduct()
deleteProduct()

## Constants

UPPER_CASE_WITH_UNDERSCORES

## Constructor Injection

Always use constructor injection.

## One Responsibility

Each class should have a single responsibility.
